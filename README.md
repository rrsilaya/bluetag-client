<h4 align="center">An open source retail shop monitoring system</h4>

### Technology Stack
* ReactJS
* Redux
* Grommet

### Installation
1. Clone this repository.
2. Install all the dependencies by running `yarn install`.
3. Run the web app using `yarn start`.
